# 🎓 Student Performance Predictor

A Machine Learning-based web application developed using **Python** and **Flask** that predicts whether a student is likely to **PASS** or **FAIL** based on Attendance Percentage and Internal Assessment Marks.

---

# 📖 Project Description

The Student Performance Predictor is a simple web application that helps predict a student's academic performance using a Decision Tree Machine Learning model. Users enter their attendance percentage and internal assessment marks, and the trained model predicts the result.

This project demonstrates the practical implementation of Machine Learning with a Flask web application and provides a simple, interactive user interface.

# ⚙ Installation

1. Download or clone the project.

2. Install the required libraries:

```
pip install flask pandas numpy scikit-learn
```

3. Run the application:

```
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000/
```

---

# 📊 How It Works

1. Open the application.
2. Navigate to the Prediction page.
3. Enter:
   - Attendance Percentage
   - Internal Assessment Marks
4. Click the **Predict** button.
5. The Machine Learning model predicts whether the student is likely to **PASS** or **FAIL**.
6. The prediction result is displayed on the Result page.
